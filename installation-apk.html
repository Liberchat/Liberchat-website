Installation Guide

1

Allow unknown sources

Go to Settings → Security → Allow installation of applications from unknown sources.

2

Download the APK

Click the download button above or scan the QR code to get the APK file.

3

Install the application

Open the downloaded APK file and follow the installation instructions.

4

Enjoy Liberchat

Launch the application and start communicating freely and securely!
